# 💱 Convertio - Currency Converter App

## 📱 Overview
Convertio is a simple and clean Android application that converts currencies between INR, USD, EUR, and JPY. It also supports Light and Dark themes for a better user experience.

---

## ✨ Features
- 💰 Currency conversion (INR, USD, EUR, JPY)
- 🔄 Real-time UI interaction
- 🌙 Light/Dark Theme toggle
- ⚡ Fast and responsive design

---

## 🛠️ Tech Stack
- Java / Kotlin
- Android SDK
- XML (UI Design)

---

## 📂 Project Structure
- `MainActivity` → Handles currency conversion
- `SplashActivity` → App launch screen
- `res/layout` → UI design files
- `res/values` → Colors, themes, strings

---

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Device
